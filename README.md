# Easy CS

The easiest way to use any coding standard in your PHP project.

<br>

## Installation

```bash
composer require easy-cs/easy-cs --dev
```

<br>

## Usage

Run Easy CS to generate a configuration file:

```bash
vendor/bin/ecs
```

Run Easy CS again o fix your code:

```bash
vendor/bin/ecs --fix
```

<br>

That's it!
